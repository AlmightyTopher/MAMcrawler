TROUBLESHOOTING:
================================================================================

Script says "No results found"?
  - Make sure Goodreads credentials are in .env
  - Check internet connection
  - Wait a few seconds and try again

qBittorrent queue looks empty?
  - Check category is "audiobooks"
  - Check qBittorrent is running
  - May be downloading already in background

Error about "Selenium" or "webdriver"?
  - Run: pip install selenium webdriver-manager
  - Then try again
