## Creating Your Own Test Magnets

### From MAM Free Section

1. **Login to MyAnonamouse**
   - Go to https://www.myanonamouse.net

2. **Search for Free Audiobooks**
   - Click "Browse Torrents"
   - Select category: "Audiobooks"
   - Add filter: "Freeleech"
   - Sort by: "Size" (ascending)

3. **Select Small Files**
   - Choose audiobooks under 50 MB
   - Click on torrent details
   - Click "Download Torrent" or copy magnet link

4. **Replace Placeholder Magnets**
   - Copy the magnet link
   - Replace placeholders in test scripts
   - Document the torrent name and size

### Example MAM Magnet Format

```
magnet:?xt=urn:btih:<40_char_hash>&dn=<torrent_name>&tr=<tracker_url>
```

**Important:**
- Hash is 40 hexadecimal characters (0-9, A-F)
- Use proper URL encoding for torrent names (spaces = +)
- Include MAM's tracker URL for proper seeding

   - Big Buck Bunny: No seeding required
