# MAM Search Guide - Missing Audiobooks

## Search Strategy

### Basic Search Format
For each series, search: `[Author Name] [Series Name] [Book Number]`

Example: `Craig Alanson Expeditionary Force 5`

### Advanced Filters
- **Quality**: Look for 64k, 128k, or 192k bitrate
- **Narrator**: Check if the same narrator as your existing books
- **Format**: Prefer M4B over MP3 for better Audiobookshelf compatibility
- **Release Date**: Prefer recent uploads for better audio quality

## General MAM Search Tips

### Before Searching
1. Note your existing books' narrators (ensure consistency)
2. Check your preferred quality level (bitrate)
3. Review file format compatibility with Audiobookshelf

### During Search
1. Filter by audiobook category
2. Sort by seeders (higher = more reliable)
3. Check upload date (recent = better quality usually)
4. Read comments for narrator/quality feedback
5. Verify book count and chapters

### After Finding
1. Note MAM torrent ID for reference
2. Add to qBittorrent watch folder via Prowlarr
3. Monitor download progress
4. Track upload ratio
5. Keep seeding until ratio meets standards

### Quality Preferences
- **Preferred:** 128k+ M4B format
- **Acceptable:** 64k M4B (if newer encoding)
- **Avoid:** Very old MP3 files (compatibility issues)
- **Check:** Same narrator as your existing copies when possible

## Known Search Issues

### Terry Pratchett - Discworld
- Books may be listed individually without numbers
- Audiobook editions have different narrators (Briggs, Robinson, etc.)
- Some compilations available
- May find unabridged vs abridged versions

### The Land - Aleron Kong
- Series metadata inconsistent on MAM
- Some books listed under alt titles
- Check narrator matches your copies
- May be under "Chaos Seeds" subseries

### The Expanse
- Complete series collections sometimes available
- Mix of narrators (Mays is standard)
- Some audiobooks quite large (100+MB)
